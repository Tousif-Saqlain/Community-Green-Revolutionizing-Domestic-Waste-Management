Step-by-step instructions you can copy and paste onto your upload page:

1. Select Files:

Click on the "Browse" button below to select the file(s) you want to upload.

2. File Types Allowed:

- Ensure that the files you are uploading are in the following formats: image.

3. File Size Limit:

- Please note that there is a maximum file size limit of 2MB for each file.

4. Upload Progress:

- Once you have selected the file(s), you will see a progress bar indicating the status of the upload process.

5. Wait for Completion:

- Please be patient while the file(s) are being uploaded. The time it takes may vary depending on the file size and your internet connection speed.

6. Error Handling:

- If you encounter any errors during the upload process, such as exceeding the file size limit or uploading an unsupported file type, please refer to the troubleshooting tips below or contact our support team for assistance.

7. Confirmation:

- Once the upload is complete, you will receive a confirmation message indicating that your file(s) have been successfully uploaded.

8. Next Steps:

- After uploading your file(s), you may proceed to click submit button.

9. Feedback:

- We value your feedback! If you have any suggestions or encounter any issues with the upload process, please let us know by contacting communitygreen@gmail.com.
